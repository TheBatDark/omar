Networking Basics

Learn the fundamentals of networking, including concepts, devices, and protocols.

Introduction to Networking

Networking involves connecting computers and devices to share data and resources. It enables communication between devices over short and long distances.

Types of Networks

LAN

A Local Area Network (LAN) connects devices in a small area like a home or office.

WAN

A Wide Area Network (WAN) spans large geographical areas, connecting multiple LANs.

MAN

A Metropolitan Area Network (MAN) covers a city or large campus.

Networking Devices

Common devices used in networking include:

  • Routers: Forward data between networks.
  • Switches: Connect devices within a LAN.
  • Modems: Enable internet access via ISPs.

IP Addressing & Subnetting

Every device on a network has an IP address for identification. IP addresses can be IPv4 or IPv6. Subnetting helps divide large networks into smaller, more efficient sub-networks.

OSI Model

The OSI model has seven layers, each responsible for different aspects of network communication:

  1. Physical Layer
  2. Data Link Layer
  3. Network Layer
  4. Transport Layer
  5. Session Layer
  6. Presentation Layer
  7. Application Layer

Common Network Protocols

  • TCP/IP: The fundamental protocol for transmitting data over networks.
  • HTTP/HTTPS: Protocols for web traffic.
  • DNS: Translates domain names to IP addresses.

Security in Networking

Network security involves protecting data and resources from unauthorized access. Common methods include firewalls, encryption, and secure protocols like SSL/TLS.
